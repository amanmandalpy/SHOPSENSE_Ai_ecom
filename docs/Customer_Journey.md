# Customer Journey

This document details the end-to-end customer journey on ShopSense AI.

## The Complete User Path

### 1. Awareness & Entry (Landing Page)
- **Trigger:** User needs a new product, saw an ad, or read a ShopSense blog.
- **Action:** Lands on `shopsense.ai`.
- **Experience:** Greeted by a clean, premium, Google-like search interface. No cluttered banners. Fast loading.
- **Options:** Use the main search bar, click a trending category, or interact with the AI assistant prompt ("What are you looking for today?").

### 2. Discovery (Search Product)
- **Action:** User types "Best ANC Headphones under 10000".
- **Experience:** Instant auto-suggestions. Search engine parses intent (Category: Electronics > Audio, Filter: Price < 10000, Feature: ANC).
- **Result:** A clean list/grid of products matching the exact criteria, sorted by relevance or rating.

### 3. Deep Dive (Open Product)
- **Action:** User clicks on "Sony WH-CH720N".
- **Experience:** Product Details Page (PDP) loads instantly. Large, high-quality images. Clean typography.
- **Information Displayed:** Aggregate rating, brief AI summary ("Why buy this?"), technical specs, and a clear price comparison table (Amazon, Croma, Reliance Digital).

### 4. Evaluation (Compare Products)
- **Action:** User clicks "Add to Compare" and adds "JBL Tune 760NC".
- **Experience:** A side-by-side sticky view appears.
- **Highlight:** Differences in specs (e.g., Battery Life: 35h vs 50h) are highlighted with green/red indicators for easy scanning.

### 5. Assistance (Read AI Recommendation)
- **Action:** User is unsure and clicks the "Ask AI" floating button.
- **Query:** "Which one has better bass?"
- **Experience:** AI instantly responds based on aggregated reviews and specs: *"The JBL Tune 760NC features 'Pure Bass Sound' and generally receives better reviews for heavy bass, whereas Sony offers a more balanced sound profile."*

### 6. Deal Hunting (Check Coupons & Price History)
- **Action:** User decides on the Sony headphones. Wants to know if it's a good time to buy.
- **Experience:** User views the embedded Price History Chart.
- **Observation:** Current price is ₹8,990. Historical lowest was ₹7,990 during Diwali.
- **Action:** User checks the "Available Offers" section. Sees a 10% HDFC card discount.

### 7. Action 1: Postpone (Set Price Alert)
- **Scenario A (Postpone):** The user feels the price might drop next month.
- **Action:** Clicks "Set Price Alert".
- **Experience:** Modal prompts for Target Price (₹7,990) and Email/WhatsApp number. User saves it (prompted to register/login for better management).

### 8. Action 2: Conversion (Visit Merchant & Purchase)
- **Scenario B (Buy Now):** User is satisfied with the current price + coupon.
- **Action:** Clicks the "Buy on Amazon - ₹8,990" button.
- **Experience:** A seamless transition screen appears: *"Taking you to Amazon to secure this deal..."* (Affiliate tracking fires in the background).
- **Result:** User purchases the product on Amazon. ShopSense earns a commission.

### 9. Retention (Return Later)
- **Action:** User returns to ShopSense next week.
- **Experience:** Homepage recognizes the returning user. Displays "Recent Searches" and "Updates on your Wishlist".
- **Notification:** "Good news! A product on your watchlist just dropped by 15%."

## Alternate Journeys
- **The Guide Reader:** Enters via SEO -> Reads "Top 10 Laptops 2026" Blog -> Clicks product -> Compares prices -> Visits merchant.
- **The Impulsive Buyer:** Enters via push notification ("Price Drop Alert") -> Opens app/web -> Instantly clicks merchant link.
- **The Direct Searcher:** Uses browser search bar directly linked to ShopSense OpenSearch -> Bypasses homepage entirely -> Goes straight to search results.
